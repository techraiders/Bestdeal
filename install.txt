Five steps to install this project on your local host WAMP Server:
	1. Clone this repository into your C:\wamp\www\
	2. Go to phpMyAdmin and create a new database 'bestdealdb'
	3. Import database from .sql file provided with this project.
	4. Go to http://localhost/ecommerce/upload/admin/
		Login to Admin Panel using username an password given below:
		4.1 Username: Admin
		4.2 Password: secret
	5. Enjoy.
